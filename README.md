# AI Study Planner

AI powered smart study planner with quiz and timetable generation.

## Features

- AI generated study timetable
- Smart quiz generation
- User authentication with OTP
- Forgot password system
- Responsive modern UI
- JWT authentication
- MongoDB integration

## Tech Stack

### Frontend
- React
- Vite
- Axios
- React Router

### Backend
- Spring Boot
- Node.js
- MongoDB
