# AI Study Planner

AI powered smart study planner with intelligent timetable generation, quiz creation, and productivity management for students.

## Live Website

[AI Study Planner Live Demo](https://ai-study-planner-rust-beta.vercel.app/?utm_source=chatgpt.com)

## Features

* AI generated study timetable
* Smart quiz generation
* Personalized study planning
* Subject based timetable creation
* Responsive modern UI
* Secure authentication system
* OTP verification
* Forgot password system
* JWT authentication
* MongoDB integration
* Fast and lightweight performance

## Installation

```bash
git clone https://github.com/your-username/ai-study-planner.git
```

```bash
cd ai-study-planner
```

```bash
npm install
```

Create a `.env` file:

```env
OPENROUTER_API_KEY=your_api_key
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

Run the project:

```bash
npm run dev
```
